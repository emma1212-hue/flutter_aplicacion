# flutter_aplicacion
Carpetas con todo el contenido de las aplicaciones creadas de ejemplo durante el curso de Flutter.
En cada carpeta se rescato los archivos importantes que contienen la base del código fuente para su funcionamiento, así mismo 
se asigno una carpeta en cada proyecto que se realizó durante el curso para almacenar una serie de capturas de como luce el programa
al momento de ejecutarlo.
